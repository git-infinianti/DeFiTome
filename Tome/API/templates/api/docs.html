üîó Blockchain Operations

GET /api/v1/blockchain/balance/{address}

Retrieve the current balance of a blockchain address across multiple chains.

Parameters:

address string Blockchain address (supports Ethereum, Bitcoin, etc.)
chain string Optional: Specific chain (eth, btc, polygon, etc.) 
{
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "balances": [
    { "chain": "ethereum", "balance": "1.5432", "usd_value": "2850.34" },
    { "chain": "polygon", "balance": "0.8921", "usd_value": "1640.22" }
  ],
  "total_usd": "4490.56"
}
GET /api/v1/blockchain/transactions/{address}

Get transaction history for a blockchain address with filtering options.

Parameters:

address string Blockchain address
limit integer Number of transactions (default: 50, max: 1000)
offset integer Pagination offset

GET /api/v1/defi/protocols

List all supported DeFi protocols with their TVL (Total Value Locked) and APY rates. 

{
  "protocols": [
    {
      "name": "Uniswap",
      "tvl": "4500000000",
      "chains": ["ethereum", "polygon", "optimism"],
      "apy_range": "3.5-25.0"
    },
    {
      "name": "Aave",
      "tvl": "5200000000",
      "chains": ["ethereum", "polygon", "avalanche"],
      "apy_range": "2.0-18.5"
    }
  ]
}
GET /api/v1/defi/pools/{protocol}

Get liquidity pools for a specific protocol with APY and liquidity information.

Parameters:

protocol string Protocol name (uniswap, aave, compound, etc.)
chain string Filter by blockchain
min_apy float Minimum APY percentage
POST /api/v1/defi/swap

Execute a token swap across multiple DEX aggregators to get the best rate.

Request Body:

from_token string Token address or symbol to swap from
to_token string Token address or symbol to swap to
amount string Amount to swap (in token units)
slippage float Maximum slippage tolerance (default: 0.5%)

üìä Market Data & Analytics

GET /api/v1/market/prices

Get real-time prices for crypto assets and tokens.

Parameters:

symbols string Comma-separated list of symbols (ETH,BTC,USDC)
currency string Base currency (USD, EUR, default: USD)
GET /api/v1/market/chart/{symbol}

Get historical price data for charting and analysis.

Parameters:

symbol string Token symbol (ETH, BTC, etc.)
interval string Time interval (1m, 5m, 1h, 1d, etc.)
from timestamp Start time (Unix timestamp)
to timestamp End time (Unix timestamp)
